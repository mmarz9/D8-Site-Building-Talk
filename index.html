Site Building is powerful!

Lets get into it...

Views in core!

Views now..

  • Handles our content management
  • Is now our default front page
  • Have the use of basic bulk operations
  • Provides us with a file manager
  • Has taken over all our Admin screens

New Field API

And some new fields

  • Date
  • Telephone
  • Link
  • Email
  • Entity Reference

WYSIWYG

Image Handling, Text Formats, Inline Editing

  • CK Editor in core
  • Tight integration with text formats
  • Image upload capabilities
  • ACF (Advanced Content Filtering)
  • Inline Editing

Content Types

  • New Layout for content creation
    • Collapsible side menu
    • “Publish” has been renamed “Promote”
  • Form Displays
    • New UI giving us greater control over the content creation form
    • Previously handled on the “Manage Field” tab in D7

Display Modes

  • View Modes
    • Are now editable and extendable
    • Can now be applied to more than just content types
  • Form Modes
    • Similar to view modes but are for forms accessible to the end user (not just content creators)
    • Currently only used on the User Registration form

Comment Types

  • Comments are now fieldable
  • We can assign different comment types to different pieces of content

Major Block Upgrades

Block Layout

  • New Layout
    • Place block button
    • Model with block filter
  • Ability to have the same block in multiple regions
  • New core/system blocks
    • Page title, Tabs, Site Branding, etc.
  • Block Types
    • Similar to content types
    • Fieldable

Configuration Management

  • Configuration is now stored in files
  • It is written while you configure the site
  • We have a new UI to manage import/export

And there is still more...

Yes, more

  • Services
  • Tour Module
  • Multilingual Support
  • Multiple image upload
  • Removal of the "upload" button

Review of some modules removed from core

  • Blog
  • PHP filter
  • Poll
  • Trigger is gone (Use rules)
  • Translation module removed. (moved to Content translation and works like entity_translation)

Review of some D7 contrib modules we no longer need

  • PLUpload
  • Features
  • Email, Phone, Date, Link, entity reference modules
  • Module filter
  • Services
  • Views, views bulk operations, admin views
  • Beans
